

## Проблема

Текущий SVG-кот выглядит как геометрическая схема — угловатый, холодный, без ощущения «пушистости». Нужен милый, округлый, пушистый котёнок с мягкими формами.

## Решение

Полностью перерисовать SVG-котёнка в `Companion.tsx`, сделав его:

### Визуальные изменения
- **Округлое тело** — овалы вместо резких форм, большая голова относительно тела (как у котят)
- **Пушистость** — волнистые контуры тела через SVG `feTurbulence` фильтр или множественные мягкие овалы по краям (имитация шерсти)
- **Большие глаза** — крупные, блестящие, с бликами (кавайи-стиль)
- **Маленький розовый нос** — треугольник → сердечко
- **Пушистый хвост** — толстый, с волнистым контуром
- **Щёчки** — розовые круги на щеках (blush)
- **Ушки** — закруглённые, с розовой внутренней частью
- **Мягкие лапки** — с подушечками (paw pads)
- **Цвет** — тёплый персиковый/рыжий вместо холодного primary, чтобы вызывать уют

### Анимации (остаются, но мягче)
- Дыхание — плавное увеличение/уменьшение тела
- Моргание — медленнее
- Хвост — плавное покачивание
- Уши — мягкие подёргивания
- Щёчки — лёгкий румянец при счастье

### Файл
| Действие | Файл |
|---|---|
| Переписать | `src/components/Companion.tsx` — новый дизайн SVG-котёнка |

Вся логика (настроение, фразы, эволюция, кормление) остаётся без изменений — меняется только отрисовка `CatCharacter`.
